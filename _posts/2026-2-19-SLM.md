For my first project for the blog, I want to get a Small Language Model running on my laptop.
If I can get an AI running locally, I won't have to use public models that require the internet. My data can be more secure!

I'm interested in Microsoft's Phi 3, because it's a lightweight model that's under the MIT license, which 
seems to be quite permissive.

But first, I'll be downloading Ollama for the interface. https://ollama.com/

Now that Ollama is downloaded, I'll just run the following in the command line:

ollama run phi3
